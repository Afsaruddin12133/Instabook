Question 3: What is the difference between local-storag and Section-storag?

Ans:

The storage are used to store data in the browser.
Local-storag:

  1. store data for long time
  2. Don't have ssl support
  3. 5 Mb limitation
  4. Data are not transferred on each HTTP request
Section-storag:
  1. store data for short time
  2. Don't have ssl support
  3. 5-10 mb limit
  4. Data are not transferred on each HTTP request
